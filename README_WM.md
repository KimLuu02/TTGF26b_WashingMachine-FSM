# Washing Machine Control Unit

The washing machine controller is a modular digital system implemented in Verilog using a finite-state-machine (FSM). The system simulates a simplified washing process and automatically progresses through different washing stages. 

The design consist of three submodules:

- UI module: Processes user inputs and validates operating conditions
- Washing Machine FSM: Controls washing states and system output
- Timer module: Provides timing control for the washing phases
